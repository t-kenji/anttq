AntTQ (Embeded Task Queue) {#mainpage}
==========================

Introduction
------------

AntTQ is Task Queue for embedded.

Dependencies
------------

- libpthread
- (optional) Catch2 (for test)

How to build
------------

```
$ make
```

How to test
-----------

```
$ make test
```

Generate doxygen document
-------------------------

```
$ make doc
```
